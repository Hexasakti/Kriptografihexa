HEXA SAKTI TUNJUNG HIDAYAT

1. simpan semua code yang kalian download pada satu folder
2. buka cmd pada komputerkamu arahkan dari cmd pindah ke folder ke tempat menyimpan filenya
3. Jalankan perintah
- javac Main.java
- javac VignereCipher.java
- javac PlayfairCipher.java
- javac HillCipher.java

4. lalu jalankan program menggunakan CMD dengan commanc
- java Main

CARA PAKAI PROGRAM
a. Pilih Jenis Cipher: Di atas ada tiga pilihan: Vigenere, Playfair, sama Hill. Klik yang mau kamu pake.

b. Masukin Pesan: Bisa langsung ketik pesannya di kotak "Masukkan Pesan", atau kalo mau pake file, klik tombol "Upload File" terus pilih file .txt-nya.

c. Masukin Kunci: Ketik kunci rahasiamu di kotak "Masukkan Kunci". Ingat ya, minimal 12 huruf!

d. Enkripsi atau Dekripsi: Kalo mau enkripsi, klik tombol "Enkripsi". Kalo mau dekripsi, klik "Dekripsi". 

e. Liat Hasilnya: Hasilnya bakal muncul di kotak "Hasil" di bawah.
